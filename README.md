# 智能数据匹配工具 - 专业匹配 & 院校匹配

## 项目简介

本项目是一个纯前端的智能数据匹配工具，集成了**专业匹配**和**院校匹配**双重功能。通过先进的多级匹配算法，能够精准处理各种复杂的匹配场景，特别适用于教育机构的数据整理和验证工作。

**核心特色：**
- 🎯 **双重匹配系统**：专业匹配（4级算法）+ 院校匹配（5级算法）
- 🔒 **数据安全**：完全在浏览器端运行，数据不上传服务器
- 🎨 **直观界面**：标签式设计，操作简单直观
- 📊 **智能分析**：实时统计和可视化结果展示
- 📁 **格式兼容**：支持Excel、CSV等多种格式

## 主要功能

### 专业匹配功能
- **4级渐进式匹配算法**：从精确匹配到模糊匹配，确保高匹配率
- **智能数据校验**：自动验证和修正学制、开学时间、申请费用等关键字段
- **格式标准化**：统一数据格式，提升数据质量

### 院校匹配功能  
- **5级渐进式匹配算法**：专为中国院校设计的智能匹配系统
- **简称识别**：自动识别"北大"、"清华"等常见院校简称
- **中英混合处理**：支持"Beijing大学"、"清华University"等混合格式
- **容错能力**：处理拼写错误、格式混乱等各种复杂情况
- **网络俗称识别**：识别"五道口技校"等网络俗称

## 匹配算法详解

### 专业匹配算法（4级）

1. **Level 1: 精确匹配**
   - 通过专业ID精确匹配
   - 置信度：1.0

2. **Level 2: 标准化匹配**  
   - 通过专业链接匹配
   - 文本标准化后匹配
   - 置信度：0.95

3. **Level 3: 关键词匹配**
   - 专业中文名 + 学位等级匹配
   - 基于Jaccard相似度
   - 阈值：≥0.6

4. **Level 4: 模糊匹配**
   - 英文名模糊匹配
   - 基于编辑距离算法
   - 阈值：≥0.5

### 院校匹配算法（5级）

1. **Level 1: 精确匹配**
   - 完全相同的院校名称
   - 置信度：1.0

2. **Level 2: 标准化匹配**
   - 去除标点、空格、大小写标准化
   - 全角半角转换
   - 置信度：0.95

3. **Level 3: 关键词匹配**
   - 提取地区、类型、核心关键词
   - Jaccard相似度计算
   - 阈值：≥0.8

4. **Level 4: 模糊匹配**
   - Levenshtein编辑距离算法
   - 包含关系检测
   - 阈值：≥0.7

5. **Level 5: 语义匹配**
   - 简称映射表匹配
   - 核心词汇相似度
   - 网络俗称识别
   - 阈值：≥0.6

## 使用方法

### 专业匹配
1. 切换到"专业匹配"标签
2. 上传现有专业数据库Excel文件
3. 上传待匹配的新专业数据Excel文件
4. 点击"开始匹配和校验"
5. 查看匹配结果和统计信息
6. 下载处理后的结果文件

### 院校匹配
1. 切换到"院校匹配"标签  
2. 上传标准院校数据库文件（包含ID、中文名、英文名）
3. 上传待匹配的院校名单文件
4. 点击"开始匹配"
5. 查看匹配结果，包含置信度和匹配级别
6. 导出匹配结果
